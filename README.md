```bash
# Clone this project
$ git clone https://github.com/certifiedTboy/file-upload-queue.git

# Access
$ cd file-upload-queue

# Install dependencies
$ npm install

# Run the project in watch mode
$ npm run start:dev (Development server)

# The server will be served on <http://localhost:3000>
```

## Environmental Variables

For the enviromental variables

- Create a file in the root folder named .env
- Reference the env-example file in the root folder of the application for the data you need to include in .env file
